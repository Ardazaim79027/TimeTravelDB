# TimeTravelDB+

## Project Description

TimeTravelDB+ is an agentic temporal database system that reconstructs historical database states and allows interactive querying of historical data.

## Features

* Change Logging Agent
* Reconstruction Agent
* Comparison Agent
* Explanation Agent
* Query Agent
* Statistics Agent
* Historical Snapshot Reconstruction
* Interactive User Interface

## Technologies Used

* Python
* FastAPI
* MySQL
* SQLAlchemy
* HTML/CSS

## How to Run

1. Install dependencies:

pip install -r requirements.txt

2. Start server:

python -m uvicorn main:app --reload

3. Open browser:

http://127.0.0.1:8000
